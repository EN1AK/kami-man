# Kami Man Bot 帮助

## 游戏王查卡

| 命令 | 说明 | 示例 |
|---|---|---|
| ck 卡名 | 查询单张卡片信息，包含卡图 | ck 青眼白龙 |
| 查卡 卡名 | ck 的中文别名 | 查卡 黑魔导 |
| 卡图 卡名 | 只查询卡图 | 卡图 灰流丽 |
| pl 卡名 | 批量查卡，返回合并聊天记录 | pl 耀圣 |
| 批量查卡 卡名 | pl 的中文别名 | 批量查卡 白龙 |
| 裁定 卡名 | 查询卡片裁定，返回合并聊天记录 | 裁定 增殖的G |


## MyCard 对战平台

| 命令 | 说明 | 示例 |
|---|---|---|
| 绑定 用户名 | 绑定自己的 MyCard 用户名 | 绑定 PlayerA |
| 查询绑定 | 查询自己的绑定信息 | 查询绑定 |
| 查询绑定 @某人 | 查询别人绑定信息 | 查询绑定 @用户 |
| 历史 | 查询自己本月历史战绩 | 历史 |
| 5月历史 | 查询自己 5 月历史战绩 | 5月历史 |
| 25年5月历史 用户名 | 查询指定用户指定月份战绩 | 25年5月历史 PlayerA |
| 胜率查询 | 查询自己的总胜率统计 | 胜率查询 |
| 胜率查询 用户名 | 查询指定用户胜率统计 | 胜率查询 PlayerA |
| 首胜查询 | 查询自己今日首胜 | 首胜查询 |
| 首胜查询 用户名 | 查询指定用户今日首胜 | 首胜查询 PlayerA |
| 订阅 用户名 | 订阅用户新对局推送 | 订阅 PlayerA |
| 订阅 | 订阅自己绑定的用户 | 订阅 |
| 退订 用户名 | 取消订阅 | 退订 PlayerA |

## 系统管理
